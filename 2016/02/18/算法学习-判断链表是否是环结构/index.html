[算法学习]判断链表是否是环结构

  • Java
算法

问题描述： 判断链表是否是环结构。

解法与分析： 用两个指针的方法判断。

  1. 第一个指针在第一个结点，第二个指针在第二个结点处；
  2. 两个指针同时移动，第一个指针每次移动一个结点，第二个指针每次移动两个结点；
  3. 当第二个指针和第一个指针指向的结点是同一个时，链表有环。
