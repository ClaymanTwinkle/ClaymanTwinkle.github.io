三、安全的动态数组Vector

加锁版的ArrayList（线程安全）。方法实现与ArrayList差不多，不同的是线程不安全的方法都加上锁（synchronized）了，这里就不讲啦。
一般不使用Vector而使用ArrayList，除非在多线程环境下（万不得已）才使用Vector。
